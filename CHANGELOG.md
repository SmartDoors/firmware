# Release v1.3.0

**Build Date:** 2026-01-07

## 변경 사항

[26-01-06]
- Factory 페이지 권한 체크 및 로그인 리다이렉트 추가
- Factory 페이지 디바이스 정보 표시 개선 (그리드 레이아웃)
- CHANGELOG 생성 로직 개선 (최신 내용이 상단에 오도록, 중복 방지)
- Version.txt에서 변경사항 추출 로직 개선 (버전별 정확한 추출, 날짜 형식 확장)
- OTA 업데이트 시 도어가 멈출 때까지 대기 기능 추가
- OTA 업데이트 로그에 실제 파일명 표시 개선
- Update 페이지에서 동일 버전일 경우 다운로드 목록 숨김 처리
