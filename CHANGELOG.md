# Release v1.3.0

**Build Date:** 2026-01-05

## 변경 사항

[25-11-03]
- OTA update 기능 추가로
- 도어 동작 관련 설정 최적화

[25-05-20]
- 도어끝단에서 전류 차단시 발생하는 유도성kickback 개선
- 도어 동작 관련 설정 최적화

[25-05-19]
- 도어끝단에서 전류 차단시 발생하는 유도성kickback 개선
- 점진적 감속 구현으로 도어 끝단에서 부드러운 정지 구현
- 소프트 스톱 기능 추가로 전압 제한을 점진적으로 감소시켜 안전한 전류 차단

[25-05-12]
- FOC_One_Sliding_Door
- 초기화시 최대 도어 길이 체크 조건 삭제
- 초기화 작동 시간 30초 -> 60초 증가
- 예가도어 데모 적용 파라미터 적용

[25-05-17]
- DoorDirection 열거형을 MotionType으로 변경하고 값 업데이트
- CommandInterface, DoorController, MotorManager 파일 삭제하여 코드베이스 간소화
- 사용하지 않는 파일 제거 및 코드 구조 개선
- WebService 클래스와 메모리 관리 최적화
- Door 열림/닫힘 동작 개선

[25-05-07]
WebServer 통합
- 열림/닫힘 버튼 동작
- 구간 열림 추가
- 간혹 웹서버 heep 메모리 크래시로 클라이언트 접속 상태에 따라 Notifiy 함수 처리 --> 추후 개선필요

## 기술 정보

- **Firmware Version:** 1.3.0
- **Hardware:** BOARD_LMC_REV_4
- **Chip:** ESP32-S3

## 설치 방법

1. OTA 업데이트 페이지 접속: `http://<ESP32-IP>/update`
2. GitHub 버전 체크에서 자동 확인
3. "최신 버전 다운로드 (로컬 저장)" 버튼 클릭
4. 수동으로 펌웨어 파일 업로드

## 주의사항

- 업데이트 중 전원을 끄지 마세요
- 도어 작동 중에는 업데이트하지 마세요
- 업데이트 후 자동으로 재시작됩니다
